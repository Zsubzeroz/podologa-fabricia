# Podóloga Fabrícia - Sistema de Gestão

![React](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)
![Vite](https://img.shields.io/badge/Vite-8-purple?style=for-the-badge&logo=vite)
![Vercel](https://img.shields.io/badge/Vercel-SPA-black?style=for-the-badge&logo=vercel)

Um sistema completo de gestão para clínicas de podologia, focado em agendamento, prontuários de pacientes, controle financeiro, compras e gerenciamento de produtos.

---

## 🚀 Funcionalidades

### 📅 Agenda & Atendimentos
- **Agendamento Prático**: Controle completo de horários para a podóloga.
- **Cancelamento e Alteração**: Fluxo simples para reagendamento.
- **Visão por Profissional e Sala**: Suporte para múltiplas salas e profissionais.

### 👥 Gestão de Clientes e Ficha de Anamnese
- **Ficha de Anamnese Detalhada**: Armazenamento completo de prontuários, condições de saúde e histórico do paciente.
- **Visualização de Vendas por Cliente**: Histórico detalhado de compras de serviços e produtos.

### 💰 Financeiro & Caixa
- **Controle de Caixa**: Abertura, fechamento e lançamentos diários.
- **Análise Mensal e Anual**: Gráficos e tabelas para visualizar o faturamento e despesas.
- **Categorias e Contas**: Classificação detalhada para receitas e despesas.
- **Formas de Pagamento**: Flexibilidade para múltiplos métodos de pagamento.

### 📦 Produtos & Compras
- **Estoque**: Controle preciso de quantidade em estoque de insumos e produtos para venda.
- **Fornecedores & Compras**: Gestão de compras com fornecedores parceiros.

---

## 🛠️ Tecnologias Utilizadas

- **React 19**: Biblioteca para construção de interfaces web reativas.
- **Vite 8**: Ferramenta de build extremamente rápida.
- **Lucide React**: Ícones premium e modernos.
- **Vercel**: Deploy automatizado com suporte a Single Page Application (SPA).

---

## 📦 Como Executar o Projeto Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/Zsubzeroz/podologa-fabricia.git
   cd podologa-fabricia
   ```

2. **Entre na pasta do frontend:**
   ```bash
   cd frontend
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Abra o navegador em:**
   ```
   http://localhost:5173
   ```

---

## 🌐 Deploy na Vercel

O sistema já está configurado para deploy na Vercel através do arquivo `vercel.json` localizado na raiz do projeto. Toda alteração na branch principal realiza o build e deploy automaticamente.
